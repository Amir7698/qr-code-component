# Frontend Mentor - QR Code Component

This is a professional solution to the [QR code component challenge on Frontend Mentor](https://www.frontendmentor.io/challenges/qr-code-component-iux_Z08S7). This project focuses on building a pixel-perfect component using a mobile-first workflow and modern CSS practices.

## Overview

### The Challenge

The objective was to build a QR code component that matches the provided design as closely as possible. 

Key features:
- Responsive layout for mobile and desktop devices.
- Implementation of precise design specifications from Figma (colors, typography, and spacing).
- Semantic HTML for better accessibility and SEO.
